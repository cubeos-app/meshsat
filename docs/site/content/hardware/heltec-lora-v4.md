# Heltec LoRa 32 V3

The Heltec WiFi LoRa 32 V3 is the recommended Meshtastic radio for MeshSat. It features an ESP32-S3, SX1262 LoRa transceiver, OLED display, and USB-C.

## Setup

1. Flash Meshtastic firmware via [Meshtastic Web Flasher](https://flasher.meshtastic.org)
2. Configure your region (EU_868, US, etc.) via the Meshtastic app
3. Connect to the Pi via USB-C — MeshSat auto-detects it

## Specifications

| Parameter | Value |
|-----------|-------|
| MCU | ESP32-S3 |
| LoRa | SX1262 |
| Frequency | 868 MHz (EU) / 915 MHz (US) |
| USB | USB-C (serial) |
| Display | 0.96" OLED |
| Battery | JST connector for LiPo |
